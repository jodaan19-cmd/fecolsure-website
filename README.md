# FECOLSURE - Página Web Institucional

Federación Colombiana de Suboficiales en Retiro

## Tecnologías
- React 18.2.0
- TailwindCSS 3.3.0
- Lucide React

## Instalación
```bash
npm install
npm start
```

## Características
- Diseño responsivo
- Formulario con validaciones
- Modal de historia interactiva
- Accesibilidad WCAG 2.1 AA
